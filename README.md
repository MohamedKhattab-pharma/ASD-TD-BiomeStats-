# ASD-TD-BiomeStats- (In conception stage)
Project on identifying and comparing microbiota biomarkers in ASD-Children vs Control.

Rough outline (4 Groups): 2 ASD patients with similar GI Disease, 2 ASD Patients without GI Disease, 2 Control Patients with similar GI Disease and 2 Control Patients without GI Disease.

Idea: Exploratory shotgun metagenomic profiling of gut microbiota in ASD and neurotypical children stratified by gastrointestinal disease status.

Methods:
Shotgun metagenomic preprocessing, Taxonomic profiling, Functional profiling, Diversity analysis, Exploratory biomarker analysis

Including:
PCA / PCoA clustering plots
heatmaps
differential abundance volcano plots
pathway enrichment comparisons
